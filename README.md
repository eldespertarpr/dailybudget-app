# DailyBudget

DailyBudget es una aplicación sencilla para controlar cuánto puedes gastar cada día sin romper tu presupuesto del periodo.

La idea principal es simple: escribes cuánto dinero tienes disponible y cuántos días debe durar. La app calcula tu presupuesto diario, registra tus gastos, separa pagos o metas protegidas, y recalcula automáticamente lo que puedes gastar.

## Funciones principales

- Calcula el dinero disponible por día.
- Registra gastos diarios.
- Permite registrar gastos atrasados.
- Muestra cuánto puedes gastar antes de pagar.
- Tiene modos de control diario:
  - Estricto
  - Normal
  - Flexible
- Permite separar pagos y metas antes de calcular el gasto libre.
- Guarda historial protegido de pagos y metas separadas.
- Muestra el estado del día:
  - Abierto
  - Cerrado hoy
- Evita cerrar el mismo día dos veces.
- Exporta e importa respaldos en archivo JSON.
- Funciona como PWA instalable en Android/Chrome.
- Guarda los datos localmente en el dispositivo.

## Instalación

Abre la app desde:

https://eldespertarpr.github.io/dailybudget-app/

En Android Chrome:

1. Abre la página.
2. Toca el menú de tres puntos.
3. Elige **Instalar app**.
4. DailyBudget aparecerá en la pantalla principal.

## Uso básico

1. Toca **Nuevo Plan**.
2. Escribe el saldo disponible.
3. Escribe la cantidad de días del periodo.
4. Toca **Comenzar Plan**.
5. Registra tus gastos durante el día.
6. Usa **¿Puedo gastar esto?** antes de pagar.
7. Usa **Plan Protegido** para separar pagos o metas.
8. Al terminar el día, toca **Terminar Día**.
9. Exporta un respaldo periódicamente.

## Privacidad

DailyBudget no usa cuentas, servidores ni sincronización en la nube.

Los datos se guardan localmente en el navegador del dispositivo usando almacenamiento local.  
El usuario puede exportar un respaldo en archivo JSON para guardar o mover sus datos.

## Respaldo

Usa **Exportar** para descargar un archivo de respaldo.

Usa **Importar datos desde archivo** para restaurar un respaldo anterior.
